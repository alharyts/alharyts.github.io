Tentang Saya

Halo! Saya adalah seorang lulusan SMK jurusan Rekayasa Perangkat Lunak (RPL) yang memiliki semangat besar dalam pengembangan web. Saya baru saja menyelesaikan pendidikan saya dan sangat antusias untuk memulai karir sebagai seorang Web Developer.
